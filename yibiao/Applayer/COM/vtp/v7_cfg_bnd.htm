  • DCAN: bus_off_auto_recovery is configurable; frame checksum support; enhance timeout.

V7BND version 10.4.0:

  • Support for Softune30L18 MB96F615 cCAN added. (Bus off auto recovery configuration is not supported by target).
  • Support for VTP 7.1 added: NVRAM checksum calculation feature added.
  • Support for frame checksum added.

V7BND version 10.3.0:

  • Support for GHS 6.1.4 V850E2S-Fx4-L AFCAN added. (Bus off auto recovery configuration is not supported by target).

V7BND version 10.2.0:

  • Support for VTP 7.1 added: NVRAM checksum calculation feature added
  • FlexCAN_vct: bus-off auto recovery is configurable, frame checksum support.

V7BND version 10.1.0:

  • Support for VTP 7.1 added: enhanced timeout and frame checksum features (both require device driver support).
  • FlexCAN: bus-off auto recovery is configurable, frame checksum support.
  • Controllers added: "cCAN", "FlexCAN_vct".
  • Targets added: "CGT PACE6T", "GCC ST32".

V7BND version 10.0.0:

  • Initial release for VTP 7.0.
